## Design & build

A modern, **dark editorial** design with electric-blue accents — kinetic typography, a custom cursor, magnetic buttons, smooth scroll-reveals and a framed portrait. Fully responsive, accessible, and motion-aware (`prefers-reduced-motion`).

Built as a lightweight **zero-build static site** with plain **HTML, CSS and vanilla JavaScript** — no framework, no dependencies — and hosted on **Vercel**.
